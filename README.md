# Flint Pay API — Bug Reports & Feedback

This repository is the public issue tracker for the [Flint Pay API](https://withflintpay.com).

**No source code is hosted here.** Use the links below to get started.

## Resources

- [API Documentation](https://withflintpay.com/docs)
- [Developer Dashboard](https://dashboard.withflintpay.com)

## Reporting a Bug

1. [Search existing issues](../../issues) to avoid duplicates
2. [Open a new bug report](../../issues/new/choose)
3. Fill out all required fields — incomplete reports will be closed

## Security Vulnerabilities

**Do not file security issues publicly.**

Report vulnerabilities privately to: security@withflintpay.com

We aim to respond within 48 hours.

## Guidelines

- One issue per bug
- Sanitize any API keys or sensitive data before posting
- Issues from accounts created less than 30 days ago may require additional review
- Spam, AI-generated noise, and low-effort reports will be closed without comment
